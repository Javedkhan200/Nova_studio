# Nova Pro Engine (v2.1) 🚀

**Nova** एक अत्याधुनिक प्रोग्रामिंग लैंग्वेज और फ्रेमवर्क है, जो सीधे टर्मिनल (Termux/Linux) से शक्तिशाली AI और OS ऑटोमेशन करने के लिए डिज़ाइन किया गया है।

## 🌟 मुख्य विशेषताएँ (Key Features)
- **AI Self-Healing:** `Nova.ai.heal()` कमांड के साथ अपना कोड खुद डायग्नोस और फिक्स करें।
- **Native OS Execution:** `Nova.os.run()` के जरिए सीधे ऑपरेटिंग सिस्टम कमांड्स चलाएं।
- **Security Tools:** इनबिल्ट `Nova.net.scan()` से नेटवर्क का विश्लेषण करें।
- **High Performance:** C में लिखा गया इंजन, जो लो-लेवल पावर और हाई-लेवल AI का तालमेल बिठाता है।

## 🛠 कैसे उपयोग करें (Quick Start)
1. **इंजन इंस्टॉल करें:** `clang -O3 nova_parser.c -o nova_lang`
2. **प्रोग्राम बनाएं:** `my_script.nova` फाइल लिखें:

3. **रन करें:** `nova_lang my_script.nova`

## 📦 Nova Studio Extension
यह Nova के लिए ऑफिशियल VS Code Extension है, जो कोडिंग के दौरान सिंटैक्स हाइलाइटिंग और ऑटो-कंप्लीशन सपोर्ट देता है।
